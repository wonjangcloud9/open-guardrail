# Phase 2: Production-Ready Ecosystem — Design Spec

## Overview

Phase 1(v0.3.0)에서 25개 가드 + 코어 엔진 완성.
Phase 2는 "사람들이 실제로 쓸 수 있는" 상태로 만드는 단계.
npm 메타데이터 정비, examples, 프리셋, 오픈소스 인프라, CI/CD 강화.

## Goals

1. npm 패키지 메타데이터 완전 정비 (검색/신뢰성)
2. umbrella 패키지가 모든 26개 가드 re-export
3. 실행 가능한 예제 6종 제공
4. 프리셋 5종 (default, strict, korean, security, content)
5. 오픈소스 기반 문서 (CONTRIBUTING, SECURITY, 이슈 템플릿)
6. CI/CD 강화 (coverage, lint, publish workflow)
7. 전 패키지 버전 v0.5.0 통일
